## Project brief

Pixel Archive 使用 Astro 和 Content Collections 管理项目、笔记、存档点与实验内容。
当前阶段重点建立稳定、克制且适合长期维护的内容展示结构。

## Current save point

- 基础视觉系统与浅色 / 深色主题已经建立
- 首页 Archive Terminal 和一级页面路由已经可用
- 四类内容集合已经完成 schema 校验
- Projects 列表、状态、技术栈、关联记录与详情页已经接入
- 真实内容公开边界已经完成清点

## Structure

项目按四类内容组织：

| Collection | Role |
| --- | --- |
| Projects | 长期项目与阶段状态 |
| Notes | 技术文章、日志与短笔记 |
| Archive | 可按时间回看的保存点 |
| Lab | 尚未固化为正式功能的界面实验 |

## Current constraints

Pixel Archive 不把私人草稿、密钥、本机路径或未经核验的项目指标写入公开内容。真实截图会在身份信息与内容脱敏完成后再加入项目记录。
