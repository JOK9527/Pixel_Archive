## Project brief

GLM-OCR Tool 将文档识别封装为本地命令行流程，支持 PDF、JPG、PNG 和 JPEG。识别完成后生成 Markdown 与 JSON，便于继续清理文本、检查公式和表格，或接入后续文档流程。

## Supported workflow

```powershell
glm-ocr "<file-path>"
glm-ocr "<file-path>" -StartPage 1 -EndPage 3
glm-ocr "<file-path>" -OutputDir "<output-dir>"
```

输出文件与输入文件同名，并分别使用 `.glm_ocr.md` 和 `.glm_ocr.json` 后缀。

## Highlights

- 使用单一命令处理 PDF、JPG、PNG 和 JPEG
- 支持页码范围与输出目录参数
- 同时保留便于阅读的 Markdown 和便于继续处理的 JSON
