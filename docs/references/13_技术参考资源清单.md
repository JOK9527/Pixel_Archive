# 13_技术参考资源清单

版本：v1.0  
项目：Pixel Archive  
位置：`docs/references/13_技术参考资源清单.md`  
用途：整理 Pixel Archive 后续可能用到的技术资料、官方文档、学习入口和参考方向。本文档仅作为技术参考库，不代表当前阶段必须使用或允许引入相关技术。  

---

## 1. 本文档的作用

本文档负责记录 Pixel Archive 可能用到的技术参考资料，包括：

```txt
Astro
TypeScript
MDX
Astro Content Collections
CSS Variables
Markdown / MDX 渲染
Cloudflare Pages
Pagefind
RSS
SEO
Open Graph
图片优化
可访问性
```

本文档不是开发规范。

是否使用某项技术，应以以下文档为准：

```txt
docs/00_项目开发计划书.md
docs/01_规范总览指南.md
docs/standards/02_技术与开发执行规范.md
docs/standards/08_部署与运维规范.md
docs/standards/09_验收与测试规范.md
```

---

## 2. 使用原则

技术参考资源只用于：

```txt
查文档
确认 API
参考最佳实践
排查问题
规划后续功能
```

不代表：

```txt
当前阶段必须引入
Agent 可以主动安装
可以改变技术栈
可以绕开项目规范
```

例如：

```txt
本文档提到 Pagefind
不代表第一阶段要实现搜索

本文档提到 remark / rehype
不代表第一阶段要引入 Markdown 插件

本文档提到 SEO / Open Graph
不代表第一阶段要做完整 SEO 系统
```

## 11. 最终原则

> 技术参考是工具箱，不是待办清单。  
> Pixel Archive 第一阶段应保持 Astro + TypeScript + MDX + Content Collections + 原生 CSS 的轻量路线；其他技术只有在确实需要时，才分阶段引入。
